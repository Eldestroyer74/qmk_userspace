# QMK keymaps

Keymap JSON file content are preprocessors to macros in `layout.h`. They can be compiled directly using `qmk compile <keyboard>.json` command.

## 3x6_3 Layout
```
,-----------------------------------------.                ,-----------------------------------------.
| TAB  |  Q   |  W   |  E   |  R   |  T   |                |  Y   |  U   |  I   |  O   |  P   | BSPC |
 ```
